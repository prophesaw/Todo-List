# 📝 Todo List (Flask)

A simple **Todo List web application** built with **Flask** using the render for deployment.  
This project lets users add, view, and manage tasks easily.

---

## 🚀 Features
- Add new tasks
- Delete tasks
- Deployable on [Render](https://render.com)
